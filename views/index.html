Timestamp Microservice based on ISO_8601

Example Usage:

  • [project url]/api/20151225
  • [project url]/api/2015-1225
  • [project url]/api/2015-12-25

  • [project url]/api/20151225T20:21:22
  • [project url]/api/2015-1225T20:21:22
  • [project url]/api/2015-12-25T20:21:22

  • [project url]/api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
